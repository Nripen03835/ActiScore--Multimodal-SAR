API Reference

Complete reference for integrating ActiScore features into your applications.

API Overview

Base URL

https://api.actiscore.com/v1

Authentication

Bearer Token in Authorization header

Response Format

All API responses are returned in JSON format with the following structure:

{
  "success": true,
  "data": { ... },
  "message": "Operation completed successfully",
  "timestamp": "2024-01-15T10:30:00Z"
}

Authentication

All API endpoints require authentication using Bearer tokens.

Get API Token

POST /auth/token
Content-Type: application/json

{
  "email": "your-email@example.com",
  "password": "your-password"
}

Using the Token

GET /api/emotion-analysis
Authorization: Bearer YOUR_API_TOKEN
Content-Type: application/json

Emotion Analysis API

Analyze Video Emotions

POST /api/emotion-analysis
Authorization: Bearer YOUR_API_TOKEN
Content-Type: multipart/form-data

video: [video file]
options: {
  "extract_audio": true,
  "detect_faces": true,
  "frame_interval": 30
}

Success Response

{
  "success": true,
  "data": {
    "dominant_emotion": "happy",
    "confidence": 0.85,
    "emotion_timeline": [...],
    "audio_emotions": [...],
    "processing_time": 12.5
  }
}

Document Summarization API

Summarize Document

POST /api/document-summarize
Authorization: Bearer YOUR_API_TOKEN
Content-Type: multipart/form-data

document: [PDF/TXT/DOC file]
options: {
  "summary_type": "executive",
  "max_length": 500,
  "language": "en"
}

Success Response

{
  "success": true,
  "data": {
    "summary": "Document summary text...",
    "key_points": ["Point 1", "Point 2", "Point 3"],
    "original_length": 1500,
    "summary_length": 200,
    "processing_time": 3.2
  }
}

Video Summarization API

Summarize Video

POST /api/video-summarize
Authorization: Bearer YOUR_API_TOKEN
Content-Type: application/json

{
  "video_url": "https://youtube.com/watch?v=...",
  "summary_type": "educational",
  "max_duration": 300,
  "extract_transcript": true
}

Success Response

{
  "success": true,
  "data": {
    "title": "Video Title",
    "summary": "Video summary text...",
    "key_points": ["Point 1", "Point 2", "Point 3"],
    "transcript": "Full transcript...",
    "duration": "15:30",
    "processing_time": 45.8
  }
}

Research Tools API

Search Research Papers

POST /api/research-search
Authorization: Bearer YOUR_API_TOKEN
Content-Type: application/json

{
  "query": "machine learning emotion recognition",
  "keywords": ["deep learning", "multimodal"],
  "max_results": 10,
  "sort_by": "relevance"
}

Success Response

{
  "success": true,
  "data": {
    "query": "machine learning emotion recognition",
    "papers": [
      {
        "title": "Deep Learning for Emotion Recognition",
        "authors": ["John Doe", "Jane Smith"],
        "abstract": "This paper presents...",
        "year": 2023,
        "relevance_score": 0.92
      }
    ],
    "total_found": 25
  }
}

Error Handling

API errors are returned with appropriate HTTP status codes and error messages.

Authentication Error

{
  "success": false,
  "error": "Invalid authentication token",
  "message": "Please provide a valid Bearer token in the Authorization header"
}

Validation Error

{
  "success": false,
  "error": "Validation failed",
  "message": "Missing required field: video_url"
}
